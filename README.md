För Week_3_4 rekomenderar jag att böja med SetArrayValues serien. Efter det fortsätt med PrintArrayValues serien. 
Dem två lär ut hur man ger arrays värden och sedan hur man hämtar dessa värderna. 

Efter dem två serierna rekomenderar jag i denna ordning
  1. ArrayAddition
  2. MathOnArray
  3. CloneArray
  4. AvrageArray
  5. SandardDeviation
  6. ArrayPatterns
  7. ArrayOperator - Denna är klurig, massa små uppgifter som pop() push()
Mina lösningar på dessa finns i mappen facit/
Rekomenderar att lösa uppgifterna med den lilla bas koden som redan finns i uppgiftera. Men om du ska göra det är det vikitgt att kunna grund principderna
  - Hur man skriver java [boilderplate]([url](https://souvikpauldev.github.io/boilerplateworld/language/java.html))
  - Hur man definerar olika typer av arrays
  - Hur man definerar en tom array av olika typer
  - Hur metoder fungerar och vad varje bit betyder (inte public, sätt allt till public static just nu).
  - Hur, vad och när man ska returnera något.

Har ni några frågor bara DMa mig på DC, snap eller fråga mig på D2. Ska försöka förklara när jag kan! :)
